个人博客
功能描述：
本系统只支持一人注册，第一个注册的用户升级为博主。支持以 Markdown 语法编写博客，匿名与登录用户可浏览博客等。

共包含9大功能：

浏览首页
阅读文章
用户管理
	用户注册
	用户登录
标签管理
	添加标签
	编辑标签
文章管理
	添加文章
	编辑文章
	删除文章
功能结构图：
本个人博客系统仅支持两个角色，一个是游客，一个是博主（只支持一个注册用户），这两个角色分别有权访问如下功能：

游客：
	浏览首页
	阅读文章
博主：
浏览首页
阅读文章
文章管理
	添加文章
	编辑文章
	删除文章
标签管理
	添加标签
	编辑标签
用户管理
	用户注册
	用户登录
